# Analysis

This directory contains notebooks for all downstream benchmarking analyses and figure generation.

## Data preparation

Two notebooks aggregate tool outputs into intermediate files consumed by the figure notebooks.
Run these before the respective figure notebooks if intermediate files are not yet present.

| Notebook | Purpose | Output |
|---|---|---|
| `hgmm_analysis.ipynb` | Aggregates hgmm correction results | `figure_2_data/` |
| `complex_analysis.ipynb` | Aggregates complex tissue correction results | `figure_3_data/` |

## Figure notebooks

| Notebook | Figure | Content | Pre-computed data included? |
|---|---|---|---|
| `PubMedFigure.ipynb` | Fig. 1B | Tool adoption trends from GEO-filtered PubMed citations | ✅ `per_tool_years.json`, `per_tool_geo_filtered.json` |
| `Figure_2.ipynb` | Fig. 2 | Inter-species contamination and per-tool removal performance across hgmm datasets | ✅ `figure_2_data/`, `interspecies_contamination/` |
| `Figure_3.ipynb` | Fig. 3 | Biological signal preservation: marker gene enrichment, classification confidence, cell type stability | ❌ Run `complex_analysis.ipynb` first |
| `Figure_4.ipynb` | Fig. 4 | Artificial count inflation by scAR and CellClear; emergence of spurious cell types | ❌ Run `Processing/` notebooks first — `.h5ad` files not included due to size |
| `Figure_5.ipynb` | Fig. 5 | Runtime benchmark (18k–172k cells) and multi-metric radar chart | ✅ All values hardcoded |

## Note on large files

Processed `.h5ad` objects required by `Figure_4.ipynb` are not included in this repository due to file size.
They are generated by the corresponding notebooks in `Processing/`. See `Processing/README.md` for details.
